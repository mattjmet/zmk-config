# zmk-config

ZMK config repo based on [urob's zmk-config](https://github.com/urob/zmk-config).  

Most of the core functionality very closely mirrors urob's, with a different alpha layout ([Canary](https://github.com/Apsu/Canary)), a few additional combos/macros and some things just moved around/tweaked for preference.

# Changes

I don't intend to keep a full list of changes, but here I will list any more significant ones.

## Stenography

A [branch](https://github.com/mattjmet/zmk-config/tree/steno) exists making use of a [PR](https://github.com/zmkfirmware/zmk/pull/962) to enable [Stenography](http://www.openstenoproject.org/) within ZMK.  
This will likely require a custom fork of [urob's own fork](https://github.com/urob/zmk) of the ZMK firmware repo - yet to be created.
